// ROLE — 01
Hardware Team

The Hardware team is the backbone of our robot. You'll work hands-on with motors, servos, structural materials, and electronics to bring our designs to life on the competition field.

What You'll Do

  • Design and physically build the robot each season
  • Work with gears, motors, linear slides, and manipulators
  • Create 3D CAD models in OnShape or Fusion 360
  • Wire and connect electrical components safely
  • Iterate and test mechanisms under competition conditions

Great If You Know

CAD (OnShape/Fusion) Power Tools Electronics Physics 3D Printing

No experience? We'll train you. Curiosity and commitment are enough.

Apply for Hardware
// ROLE — 02
Software Team

The Software team writes the code that makes our robot intelligent. From fully autonomous routines to driver-controlled periods, you'll be responsible for turning hardware into a smart, competitive machine.

What You'll Do

  • Write autonomous and TeleOp programs in Java
  • Integrate sensors: encoders, IMUs, color sensors, cameras
  • Implement computer vision for game element detection
  • Tune PID controllers for precise movement
  • Debug code under real competition pressure

Great If You Know

Java Python Android Studio Git Arduino/Pi Algorithms

Know any coding language? You're already ahead. We'll teach the rest.

Apply for Software

// 04 — The Competition

What is FTC?

FIRST Tech Challenge is a global robotics program where student teams of up to 15 members design, build, program, and operate robots to compete in an alliance-based game. Each season brings a brand-new challenge — teams have weeks to engineer a robot capable of competing locally, regionally, and at the World Championship.

01
Design
Teams analyze the season game and sketch robot concepts. CAD models are created before a single part is cut or printed.
▼
02
Build
Hardware engineers assemble the physical robot — motors, servos, frames, and mechanisms — tested against the real game field.
▼
03
Code
Software engineers program autonomous routines and driver controls. Sensors, cameras, and PID loops make the robot smart.
▼
04
Compete
Teams face off at local qualifiers, then advance to regionals and — if they're elite enough — the FTC World Championship.
